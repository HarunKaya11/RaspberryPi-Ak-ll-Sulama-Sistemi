# RaspberryPi-Ak-ll-Sulama-Sistemi
Bu proje Dokuz Eylül Üniversitesi Yönetim Bilişim Sistemleri 2.Sınıf dersi olan Bilgisayar Donanımı dersinde RaspberryPi4 ile yaptığım akıllı saksı projesinin kodlarını ve bağlantı şemasını içermektedir şema, profesyonel programların ücretli olması nedeniyle paintten el ile çizilmiştir ama gayet anlaşılır biçimdedir.
